# 💸 Expense Tracker

A modern Expense Tracker web application built to manage income and expenses efficiently.

Users can add transactions, view total balance, track income and expenses, and filter transaction history with a clean and responsive interface.

---

## 🚀 Live Features

- Add new transactions
- Display total balance
- Show total income
- Show total expenses
- Filter transactions
- Delete transactions
- Responsive design
- Simple and clean UI

---

## 🛠️ Technologies Used

- React
- TypeScript
- Vite
- TailwindCSS
- HTML

---

## 📂 Installation and Setup

- git clone https://github.com/CodeWith-Sanjay/Expense-Tracker.git
- cd expense-tracker
- npm install
- npm run dev
